# Pertemuan 9 — README

Deskripsi singkat
-----------------

Notebook `pert9.ipynb` berisi materi dan latihan untuk Praktikum Kecerdasan Buatan (Algen) — Pertemuan 9. Notebook memadukan penjelasan teori singkat, contoh kode Python, dan visualisasi hasil eksperimen/soal.


Prasyarat
---------

- Python 3.8+ (direkomendasikan 3.8–3.11)
- Paket Python umum: `numpy`, `scipy`, `matplotlib`, `pandas`, `sympy`, `jupyter`

Instalasi cepat
--------------

1. (Opsional) Buat virtual environment:

```bash
python -m venv .venv
# Windows
.venv\Scripts\activate
# macOS / Linux
source .venv/bin/activate
```

2. Pasang dependensi singkat:

```bash
pip install numpy scipy matplotlib pandas sympy jupyter
```

Cara menjalankan
----------------

- Buka Jupyter Notebook atau JupyterLab di folder ini:

```bash
jupyter notebook
# atau
jupyter lab
```
- Buka `pert9.ipynb` dan jalankan sel secara berurutan (Run → Run All).

Catatan
------

- Jika notebook menggunakan dataset atau file tambahan, letakkan file tersebut di folder yang sama atau perbarui path pada sel yang relevan.
- Jika ada error terkait versi paket, coba gunakan versi stabil (mis. `numpy==1.25.*`, `scipy==1.11.*`).
